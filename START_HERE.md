# START HERE — Vibe Coding OS (devos layout)

If you’re new, start here. This repo stores the **system/OS** under `devos/` and keeps **your code** in the repo root.

## 1) Daily start (from repo root)
```bash
make start
```

## 2) Run Claude triage
```bash
make copy-claude
```

## 3) Start builders in parallel
```bash
make copy-codex
make copy-gemini
```

## 4) Pre‑PR check
```bash
make pr-check
```
